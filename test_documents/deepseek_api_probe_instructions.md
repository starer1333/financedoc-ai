# DeepSeek API 接入测试说明

## 上传文件

在 https://financedoc-ai.streamlit.app/ 的“数据输入与样例场景”区域，同时上传：

- `deepseek_api_probe_memo.pdf`
- `deepseek_api_probe_financials.csv`

PDF 备忘录使用英文是为了让 PyPDF2 稳定抽取文字；你仍然可以用中文向网站提问。

## 提问

在“RAG 问答助手”输入这个问题：

```text
请根据我上传的测试文档，完成 DeepSeek API 接入验证。请输出验证暗号、最主要的三个财务风险，并判断这是现金流质量问题还是收入规模问题。
```

## 如何判断 API 是否接入

如果 DeepSeek API 正常接入，回答通常会：

- 明确输出暗号：`DS-PROBE-7429`
- 能综合 PDF 里的业务描述和 CSV 里的财务数据
- 能判断主要问题是现金流质量和回款恶化，而不只是收入规模下降
- 不出现“未配置”“调用失败”“回退 demo mode”“回退本地规则”等字样

如果 API 没接上，网页仍能用本地规则回答，但回答通常会更像证据片段摘录，或者出现 API 未配置/调用失败/回退提示。
